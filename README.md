# qchat
Press <kbd>Q</kbd> to chat with LLMs in terminal.

## Install

```bash
$ npm install -g qchat
```

## Usage

```bash
# bash
$ echo 'export OPEN_API_KEY=\'<YOUR_API_KEY>\'' >> ~/.bashr
$ source ~/.bashrc

# zsh
$ echo 'export OPEN_API_KEY=\'<YOUR_API_KEY>\'' >> ~/.zshrc
$ source ~/.zshrc

# start chat
$ q
```
